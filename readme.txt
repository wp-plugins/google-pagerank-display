== Description ==

Google PageRank Display shows your site pagerank instantly and you can change style from General Options.

 * Activate the plugin
 * Go General Options [`options-general.php?page=gpdisplay`]
 * Choose style you like
 * Update option
 
 * And add this code your theme // `<?php show_gpdisplay() ?>`

Features:

* Real Google Pagerank : Shows real google pagerank, not fake
* Many Style : There are 134 different pagerank button and badges
* Automatic : just you change style from option
* Fast : Loading pagerank fast, get image remotely.
* Get your site page rank instantly

Details: `http://www.getrank.org`

== Installation ==

1. Unzip the files 
1. Upload "google-pagerank-display" directory to the `/wp-content/plugins/` directory
2. Activate the plugin from the 'Plugins' menu in WordPress
3. (optional) Change style from `Settings > GP Display`

 * And add this code your theme // `<?php show_gpdisplay() ?>`
